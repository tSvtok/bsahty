Real-time messaging non-functional for end users
Laravel Reverb is configured but WebSocket delivery, read receipts, typing indicators and unread counts are all missing. Chat is a static skeleton.

high — should fix before launch
HIGH
Token expiration causes silent 401s — users get logged out unexpectedly
No refresh token mechanism. The axios interceptor catches 401s and redirects to login, but there's no silent re-auth attempt first.
HIGH
No email verification or password reset flow
Users register with any email and have no recovery path. Missing: verification mails, reset tokens, welcome emails, mail queue jobs.
HIGH
PostCard like/reaction may silently fail (API payload mismatch)
Reaction endpoint expects a polymorphic type field. Frontend FeedPage handleLike() may be sending an incorrect payload structure.
HIGH
Authorization ownership checks incomplete in controllers
Explicit role checks were added to QuestionController and CommentController but other mutating endpoints (EventController, SpotController) lack full ownership guards — any authenticated user may edit others' data.
HIGH
No rate limiting or caching anywhere on the API
All 40+ endpoints are unthrottled. Repeated identical requests hit PostgreSQL every time. Open to brute force and DDoS.
HIGH
CORS config needs hardening before production
cors.php was created but allowed origins may still be too permissive. Needs explicit whitelist matching the production frontend URL only.
HIGH
Edit / delete own post not implemented
Users can create posts but no edit or delete UI exists. Soft deletes not configured on Question model. UpdateQuestionRequest was created but the full flow is incomplete.
HIGH
Event RSVP / join functionality missing
Events show a participant count but users cannot join or leave. RSVP state is not tracked in the database.

Badge color contrast too low — fails WCAG AA
Sport category badges use rgba(249,115,22,0.12) background — extremely light, unreadable. Focus states missing on all interactive elements.

Date formatting inconsistent across components
Some components show raw ISO strings, others format them. No shared date utility used consistently.
MEDIUM
Hover states not uniform across UI components
Mix of shadow values and transform scales. Secondary button hover only changes border color, lacks background fill change.
MEDIUM
Pagination metadata not fully consumed in frontend
Backend now paginates at 15 items. Frontend doesn't render page controls or infinite scroll, so only the first page is ever shown.
MEDIUM
No API documentation (Swagger / OpenAPI)
40+ endpoints undocumented. Slows onboarding of any new developer and makes frontend-backend contract implicit.
